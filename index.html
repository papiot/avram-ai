Activities creating the bills

I have started many hobby projects and finished very few. Here are some more notable ones
  • Mid-2015 PhobiaOVR A complete solution for self-treating phobias using Virtual Reality. We created a prototype, together with a Psychotherapist, that would allow people to get treatment for phobias using a Virtual Reality headset. We got as far as competing in Berlin for funding, unfortunatelly we did not make top 10 and decided to stop putting our own money in this. I learned a lot about the medical field and it was fun creating the prototype. More info can still be found here and here
  • Mid-2016 Emvix We wanted to create a product that can integrate Hololens (and other augmented reality devices) and remote worker help. We went as far as creating a prototype of something that now Facebook calls "Meta". It was really fun to work on this and got a first glimpse into AR and Hololens (which I think was way ahead of its time in 2015/2016). The site is still up and can be viewed here
  • When I was in highschool (around 2000) I got into an argument with someone on mIRC and they DoS me big time. I decided to make my own firewall, so I created, from an older PC a linux based hardware firewall. I used whatever version of Debian was available then, installed two network cards and wrote a routing table. I could still get attacked, but at least my real computer was protected from a direct attack
  • In 2002 I wanted to create my own version of Mathematica or Maple software. So I wrote a Matrix Operations command line tool where you can do all the basic matrix operations on multi-dimentional matrices. The original Sourceforget page is still here.
